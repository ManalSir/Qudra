# Qudra

Qudra is a bilingual RTL web platform for Qudra Innovation Center.

The public website presents Qudra as a center for innovation, science, and technology in Beersheba, working with schools, municipalities, and communities in the Negev.

The project will also include a secure internal staff dashboard for managers and instructors.

## Live Website

- Arabic: [https://qudra-center.vercel.app/ar](https://qudra-center.vercel.app/ar)

- Hebrew: [https://qudra-center.vercel.app/he](https://qudra-center.vercel.app/he)
